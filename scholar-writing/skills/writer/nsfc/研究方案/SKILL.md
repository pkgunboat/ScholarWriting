# 研究方案 Writer

## 任务

撰写国自然申报书的「研究方案」部分。这是回答"如何做"的核心章节，也是体现申请人研究深度的关键。

你将收到：
1. 该章节的大纲（来自 planning/outline.md）
2. 该章节的评审标准 checklist（写作时请对照）
3. 已完成前序章节的摘要（200-300 字/章）
4. 相关素材片段（来自 planning/material_mapping.md）
5. 论点注册表（planning/claim_registry.md）
6. 研究内容全文（本章必须与研究内容一一对应）

## 写作策略

### 推荐章节结构

#### 1. 拟采取的技术路线（1段 + 路线图）

**文字描述**：
- 总述整体技术路线的架构（输入→处理流程→输出）
- 描述各研究方向如何在技术层面串联
- 明确关键输入、输出、方法名称和评估指标

**技术路线图描述**：
- 使用 Mermaid 格式描述技术路线图结构
- 输出为 `<!-- MERMAID: ... -->` 代码块，供后续渲染
- 图文严格对应：图中的每个步骤都有文字说明，文字中的每个方法都在图中

#### 2. 各研究内容对应的技术方案（3-4节）

与研究内容一一对应，每节包含：

**（a）方法原理简述**
- 具体方法名称和核心原理（2-3句）
- 为什么选择此方法（1句）

**（b）数据来源与预处理**
- 数据集名称、规模、获取方式
- 预处理流程和标准

**（c）模型/算法设计要点**
- 核心架构/算法描述
- 关键创新点说明
- 如果涉及公式：公式前有推导说明，公式后有符号解释，不孤立呈现

**（d）评估方案**
- 评估指标（具体名称和计算方式）
- 对比基线方法
- 实验设置

**（e）可行性融入（1-2句）**
- "上述技术已在本课题组前期研究中成功应用"或
- "该方法的可行性已被***等[ref]的工作所证实"

### 层次编号体系

- 主标题：3.1、3.2（大方向，与研究内容对应）
- 一级子标题：（1）（2）（3）（技术维度）
- 二级子标题：①②③（具体技术步骤）

### 公式与文字协调（理工科）

- 公式前有推导说明（"将***建模为***问题"）
- 公式后有符号解释（"其中，$x$ 表示***"）
- 递进模式：基础定义式 → 演化约束式 → 最终优化式

#### 3. 年度研究计划

与研究内容条目严格对应：

**四年期（面上项目）**：
- 第1年：完成研究内容1（基础方法），初步开展研究内容2
- 第2年：完成研究内容2（核心技术），初步开展研究内容3
- 第3年：完成研究内容3（应用验证），整理阶段成果
- 第4年：补充实验，撰写论文，结题

**三年期（青年基金）**：
- 第1年：完成体系搭建 + 研究内容一主体
- 第2年：完成核心实验 + 研究内容二和三主体
- 第3年：补充验证 + 数据整理 + 撰写论文 + 结题

**禁写内容**："购买试剂""查阅文献""学习***技术"等无意义内容。

#### 4. 预期研究成果

- 论文发表计划（SCI/EI期刊、会议，每年分列）
- 专利/软著（如适用）
- 培养研究生计划

### 字数与篇幅

- 面上项目：5000-8000字（3-5页，不含路线图）
- 青年基金：3000-5000字（2-3页）

### 关键要求

- **具体到可操作**：评审人能判断"这个方案我自己也能照着做"
- **方案与内容一一对应**：研究内容提出的每个问题，方案中都有明确的解决路径
- **备选方案**：对可能遇到的技术困难，至少为1-2个关键步骤提供备选方案
- **图文一致**：技术路线图与文字描述严格对应

## 输出

将完成的章节写入 `sections/03_研究方案.md`

如果生成了技术路线图描述，同时输出 `figures/technical_roadmap.mmd`（Mermaid格式）。

## 约束

1. 引用 claim_registry 中的论点表述，与其他章节保持一致
2. 遵循 `references/STYLE_GUIDE_ZH.md` 中的中文学术写作规范
3. 不捏造数据、引文或实验结果——所有数据必须来自用户提供的素材
4. 避免 AI 写作痕迹（参考 `references/DEAI_PATTERNS_ZH.md`）
5. 写作时对照 checklist，确保覆盖所有 critical 和 high 级别的标准
6. 句式参考 `references/patterns/00_通用.md` + `references/patterns/04_研究方案.md`
7. 结构参考 `references/NSFC_STRUCTURE_ZH.md` 第6节

# 中文学术写作通用句式库

本文档提供跨章节复用的"语言零件"——转折、递进、强调、引出不足、引出方案等通用句式模板。供所有 Writer Agent 在写作时按需引用。

与 `NSFC_STRUCTURE_ZH.md`（章节结构规范）互补：本文档解决"怎么措辞"的问题，结构规范解决"怎么组织"的问题。

句式中 `***` 表示需替换为具体内容。

## 1. 开头引入句式

- `所谓***，是指***，其基本特征包括***和***。`
- `***通常通过***方式实现，其核心在于***。`

- `深入揭示***的***机制，对于理解***具有重要的理论意义。`

## 3. 引出不足/矛盾句式

### 3.1 指出知识空白

- `然而，***的***机制至今尚不清楚。`
- `上述研究虽从不同角度探讨了***的***问题，但***至今仍不清楚。`

- `尽管***取得了重要进展，但***问题仍未得到有效解决。`
- `以上工作为***提供了重要参考，但仍存在以下不足：***。`

### 3.3 研究深度不足

- `目前***的研究主要局限于***层面，对***缺乏系统深入的认识。`

### 3.4 研究盲区

- `值得注意的是，上述研究均未涉及***的***问题。`

### 3.5 方法局限性

- `现有***方法在***条件下表现良好，但在面对***时往往失效。`
- `该类方法主要 ***，难以应对 ***，缺乏 ***。`
- `然而，现有方法主要依赖 ***，难以 ***，在 *** 方面存在明显不足。`

### 3.6 总结性过渡

- `这些局限性严重制约了***技术/理论的进一步发展。`
- `因此，当前***技术很难应用于面向***的研究，特别是缺乏针对***的有效方法。`

## 10. 研究基础句式

## 17. 段落衔接速查表

| 段落关系 | 衔接方式 | 关键词 |
|---------|---------|--------|
| 背景→问题 | 问题对立 | 然而、伴随着 |
| 问题→回应 | 态度回应 | 面对、世界各国均高度重视 |
| 回应→不足 | 不足指出 | 然而、仍 |
| 不足→细化 | 具体细化 | 主要体现在以下几个方面 |

| 方案→意义 | 意义总结 | 综上、具有重要的科学意义与应用价值 |
| 内容间递进 | 逻辑承接 | 在此基础上、进一步地 |
| 引文→空白 | 空白引出 | 亟待解决、有必要 |

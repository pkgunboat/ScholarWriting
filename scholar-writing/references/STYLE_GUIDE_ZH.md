# 中文学术写作风格指南

本指南为中文学术写作提供句式、段落、用语、标点和衔接方面的规范，供 Reviewer Agent 在审稿时参考。

---

## 1. 句式规范

### 1.1 短句优先

每句话不超过 40 字。过长的句子应拆分为多个短句。

**反面示例：**

> 基于多模态预训练模型在跨模态理解和生成任务中所展现出的强大能力，研究者们开始探索将其应用于图形用户界面自动化操作领域的可行性。

**正面示例：**

> 多模态预训练模型在跨模态任务中表现优异。研究者开始探索其在 GUI 自动化领域的应用。

### 1.2 主动语态为主

被动语态仅在需要强调动作承受者时使用。

| 避免 | 改为 |
|------|------|
| 实验结果被记录在表 1 中 | 表 1 记录了实验结果 |
| 该方法被广泛应用于… | 研究者广泛采用该方法… |

### 1.3 避免多层嵌套定语

多层嵌套的定语从句会严重降低可读性，应拆分为多句。

**反面示例：**

> 基于视觉语言模型的针对移动端图形用户界面的自动化操作方法

**正面示例：**

> 该方法面向移动端图形用户界面，基于视觉语言模型实现自动化操作。

### 1.4 一句一义

每句话只表达一个核心信息。如果一句话中包含两个独立论点，应拆为两句。

## 2. 段落结构

### 2.1 一段一论点

每个段落围绕一个核心论点展开，不混杂多个话题。

### 2.2 首句点题

段落首句为 topic sentence，概括本段核心观点。读者只读首句就能把握段意。

### 2.3 展开论证

中间句子提供证据、数据、示例或推理，支撑首句论点。

### 2.4 尾句承启

段落末句总结本段要点，或为下一段内容做铺垫，实现段落间的自然过渡。

### 2.5 段落长度

建议每段 150-300 字。过短则论证不充分，过长则重点不突出。

## 3. 学术用语规范

### 3.1 使用精确动词

| 模糊动词（避免） | 精确动词（推荐） |
|------------------|------------------|
| 进行了实验 | 设计并执行了实验 |
| 开展了研究 | 分析了 / 调查了 / 验证了 |
| 实现了功能 | 构建了 / 提出了 / 设计了 |
| 做了对比 | 对比了 / 评估了 |

推荐动词库：提出、验证、表明、揭示、证实、量化、评估、分析、构建、推导。

### 3.2 避免口语化表达

| 口语化（避免） | 学术化（推荐） |
|---------------|---------------|
| 其实 | 实际上 / 事实上 |
| 说白了 | 具体而言 |
| 总之 | 综上 |
| 比较好 | 表现更优 |
| 差不多 | 基本一致 / 近似 |

### 3.3 避免绝对化表述

| 绝对化（避免） | 限定性（推荐） |
|---------------|---------------|
| 完全解决了 | 在…条件下有效缓解了 |
| 绝对优于 | 在…指标上优于 |
| 毫无疑问 | 实验结果表明 |
| 必然导致 | 在多数情况下会导致 |

### 3.4 数据表述要具体

| 模糊表述（避免） | 具体表述（推荐） |
|-----------------|-----------------|
| 显著提升 | 提升了 15.3% |
| 大幅下降 | 下降了 0.12 个百分点 |
| 远超基线 | 超过基线方法 8.7 个点 |

## 4. 标点符号规范

### 4.1 中英文标点区分

- 中文正文使用中文标点：，。；：、""''
- 括号内容为英文时使用英文括号：(Transformer)、(Liu et al., 2023)
- 引用外文文献名称使用英文引号："Attention Is All You Need"

### 4.2 常见错误

| 错误用法 | 正确用法 |
|---------|---------|
| 准确率,召回率 | 准确率、召回率 |
| 如图(1)所示 | 如图 1 所示 |
| Liu等人 | Liu 等人 |
| 基于Transformer的 | 基于 Transformer 的 |

### 4.3 省略号

避免连续使用省略号。学术写作中省略号的使用场景有限，多数情况下可用具体表述替代。

---

## 5. 过渡与衔接

### 5.1 逻辑过渡优先

段落间过渡应依靠内容的逻辑关系（因果、对比、递进、并列），而非机械地使用连接词。

**机械过渡（避免）：**

> 首先，我们介绍了背景。其次，我们分析了问题。最后，我们提出了方案。

**逻辑过渡（推荐）：**

> GUI 操作自动化面临两个核心挑战：界面元素的准确识别和操作序列的合理规划。准确识别依赖于…（自然过渡到下一话题）

### 5.2 连接词的学术化

如确需连接词，选择学术化程度更高的表达：

| 口语化（避免） | 学术化（推荐） |
|---------------|---------------|
| 另外 | 此外 |
| 所以 | 因此 / 据此 |
| 但是 | 然而 / 不过 |
| 而且 | 并且 / 同时 |
| 还有 | 此外 / 另外 |

### 5.3 过渡技巧

- **回指法**：用上段的关键概念作为下段的起点
- **设问法**：在上段末尾提出问题，下段回答
- **对比法**：上段讨论现有方法，下段指出不足并引出新方案
